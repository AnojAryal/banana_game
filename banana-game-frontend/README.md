
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/AnojAryal/banana_game.git
   ```

2. Navigate to the project directory:

   ```bash
   cd banana-game-frontend
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

## Development

To start the development server, run:

```bash
npm run dev
```
